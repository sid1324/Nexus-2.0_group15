# @nexus-ds/theme

Nexus DS theme primitives, here you can find the TailwindCSS `plugin`, `colors`, `layout`, `spacing`, `typography` and some other utilities.

## Installation

```sh
yarn add @nexus-ds/theme
# or
npm i @nexus-ds/theme
```
