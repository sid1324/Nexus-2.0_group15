# @nexus-ds/navbar

A responsive navigation header positioned on top side of your page that includes support for branding, links, navigation, collapse menu and more.

navbar

## Installation

```sh
yarn add @nexus-ds/navbar
# or
npm i @nexus-ds/navbar
```
